# BayesSentinel
Package for simulation and classification of multi-dimensional spectroscopic data

## Data
The data is obtained from the [Sentinel-2 satellite] (http://www.cesbio.ups-tlse.fr/us/index_sentinel2.html)

## Environment
R (>= 3.0.2)

## Dependencies
- stats
- graphics
- h5
- methods
- mvtnorm
- matrixcalc
